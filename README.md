# Запрошення на весілля — мобільна версія

Файли сайту в корені проекту. Щоб переглянути локально, просто відкрийте в браузері файл `index.html`.

Швидкий старт:

```bash
# Відкрийте файл у браузері (macOS)
open index.html
```

Файли:
- index.html — основна сторінка
- styles.css — стилі (mobile-first)
- script.js — просте збереження RSVP в localStorage

Я можу:
- Поставити ваші реальні імена/дату/місце
- Додати фото або фон
- Підключити форму, що відправляє відповіді на пошту
# va

## RSVP: резервний сервіс

У формі додано подвійне відправлення:
- спочатку через Web3Forms;
- якщо він не налаштований або недоступний, тоді через FormSubmit.

Щоб увімкнути Web3Forms:
1. Створіть безкоштовний ключ на web3forms.com.
2. Вставте ключ у [script.js](script.js#L3) замість `PASTE_WEB3FORMS_ACCESS_KEY_HERE`.

Після цього форма матиме резервний канал відправки і буде стабільнішою.
